Balanced Ternary Calculator

Balanced ternary is a way of representing numbers using the digits -1, 0, and 1. Including a negative digit can be counterintuitive at first, but gives balanced ternary some nice properties, such as

  • Can represent both positive and negative numbers without using a sign symbol
  • Rounding is equivalent to truncation
  • Ternary is the most economic integer radex for some applications
For a more detailed exploration of the properties of balanced ternary, see the wikipedia page.

Notation

For this calculator, the symbols -, 0, and + are used for -1, 0, and 1, as they are easy to type. For example, the decimal number 15 is written as +--0 in balanced ternary, since

16 = (+1)*33 + (-1)*32 + (-1)*31 + (0)*30
